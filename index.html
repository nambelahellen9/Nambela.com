Interactive Web Development Tutorial

1. HTML Basics

HTML (HyperText Markup Language) structures the content of web pages. Below is a basic structure of an HTML document.

HTML Structure Example

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
</head>
<body>
    <h1>Welcome to My Web Page</h1>
    <p>This is a paragraph of text.</p>
</body>
</html>

Exercise: Create a Basic HTML Page

Follow the steps below to create your own HTML page:

  1. Create a new file named index.html.
  2. Use the structure provided above to set up your HTML file.
  3. Add a <h2> tag with the text "My First Heading" and a <p> tag with some text.

2. CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. Below is an example of CSS styling.

CSS Styling Example

body {
    font-family: Arial, sans-serif;
    margin: 20px;
}

h1 {
    color: #007BFF;
}

p {
    color: #333;
    line-height: 1.6;
}

Exercise: Style Your HTML Page

Apply CSS styling to your HTML page as follows:

  1. In your index.html file, add a <style> section inside the <head> tag.
  2. Style the <h1> and <p> tags using CSS.
  3. Ensure the page has a consistent font and color scheme.

3. JavaScript Basics

JavaScript adds interactivity to web pages. Below is an example of a simple JavaScript function.

JavaScript Example

<script>
function showAlert() {
    alert('Hello, World!');
}
</script>

Exercise: Add Interactivity

Add a JavaScript function to your HTML page:

  1. Include a <button> element in your HTML file.
  2. Use JavaScript to display an alert with a message when the button is clicked.
